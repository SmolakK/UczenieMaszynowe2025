Uczenie Maszynowe - Wyk≈Çad 5#

Klasteryzacja to technika uczenia maszynowego bez nadzoru, kt√≥ra s≈Çu≈ºy do grupowania podobnych obiekt√≥w w zbiory na podstawie ich cech. W przeciwie≈Ñstwie do uczenia nadzorowanego, klasteryzacja nie korzysta z oznaczonych danych (brak etykiet klas) ‚Äì model samodzielnie szuka struktur i wzorc√≥w w danych.

G≈Ç√≥wne cechy klasteryzacji:

Uczenie bez nadzoru: Brak etykiet klas dla danych wej≈õciowych.
Cel: Odkrycie ukrytych struktur w danych.
Zastosowanie: Identyfikacja grup (klastr√≥w) w zbiorze danych.

Typowe algorytmy klasteryzacji:

KMeans: Algorytm iteracyjny, kt√≥ry minimalizuje odleg≈Ço≈õci punkt√≥w od centr√≥w klastr√≥w.
DBSCAN: Algorytm gƒôsto≈õciowy, kt√≥ry identyfikuje klastry jako grupy gƒôsto upakowanych punkt√≥w.
Hierarchiczna klasteryzacja: Tworzenie hierarchicznej struktury klastr√≥w w postaci drzewa.

K-Means#

To czƒôsto u≈ºywany i wzglƒôdnie prosty algorytm klasteryzacyjny. W nim, ka≈ºdy klaster jest reprezentowany przez sw√≥j centroid. Algorytm dzia≈Ça w spos√≥b iteracyjny:

Inicjalizuj losowo K centroid√≥w.
Przypisz ka≈ºdy punkt do najbli≈ºszego centroidu.
Zaktualizuj centroidy, obliczajƒÖc ≈õredniƒÖ punkt√≥w w ka≈ºdym klastrze.
Powtarzaj, a≈º centroidy ustabilizujƒÖ siƒô.

Algorytm jest prosty, szybki, ale wymaga okre≈õlenia liczby klas K z g√≥ry.

2. Klasteryzacja hierarchiczna#

Metoda ta buduje dendrogram - drzewo przedstawiajƒÖce podobie≈Ñstwo obiekt√≥w. Wyr√≥≈ºniamy dwa typy:

  • Aglomeracyjne: PoczƒÖc od ka≈ºdego obiektu bƒôdƒÖcego w jednej, osobnej klasie - ≈ÇƒÖczy je.

  • Deglomarycyjne: Zaczyba od wielkiego klastra, kt√≥ry jest stopniowo dzielony.

Zalety:

Nie ma potrzeby okre≈õlania liczby klastr√≥w
Zapewnia hierarchiƒô klastr√≥w

Wady:

Obliczeniowo obciƒÖ≈ºajƒÖcy

import numpy as np
import matplotlib.pyplot as plt

# Perform hierarchical clustering

# Plot clusters
plt.figure(figsize=(8, 6))
plt.scatter(X[:, 0], X[:, 1], c=clusters, cmap='viridis', s=50)
plt.title("Clusters Identified by Hierarchical Clustering")
plt.xlabel("Feature 1")
plt.ylabel("Feature 2")
plt.show()

3. DBSCAN (Density-Based Spatial Clustering of Applications with Noise)#

Grupuje punkty, kt√≥re sƒÖ zbli≈ºone do siebie na podstawie progu. Odfiltrowuje punkty, kt√≥re uwa≈ºane sƒÖ za ‚Äúszum‚Äù.

Kluczowe parametry: eps: Maksymalny dystans miƒôdzy punktami, kt√≥ry dopuszczony jest w klastrze. min_samples: Minimalna liczba puntk√≥w w klastrze.

Pros:

Nie wymaga okre≈õlenia liczby klastr√≥w
Odkrywa klastry o nieregularnych kszta≈Çtach
Odporny na warto≈õci odstajƒÖƒáe

Cons:

Czu≈Çy na swoje hiperparametry

import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_blobs
from sklearn.cluster import DBSCAN

# Generate synthetic data
X, _ = make_blobs(n_samples=300, centers=4, cluster_std=0.6, random_state=0)

# Apply DBSCAN
dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)

# Plot the results
plt.figure(figsize=(8, 6))
unique_labels = set(labels)
colors = [plt.cm.tab10(each) for each in np.linspace(0, 1, len(unique_labels))]

for label, color in zip(unique_labels, colors):
    if label == -1:
        # Black used for noise
        color = [0, 0, 0, 1]
    
    cluster_points = X[labels == label]
    plt.scatter(cluster_points[:, 0], cluster_points[:, 1], c=[color], s=50, label=f'Cluster {label}' if label != -1 else 'Noise')

plt.title("DBSCAN Clustering")
plt.xlabel("Feature 1")
plt.ylabel("Feature 2")
plt.legend()
plt.show()

Ocena klasteryzacji - dostƒôpny prawid≈Çowy podzia≈Ç#

Podczas oceny wydajno≈õci klasteryzacji niekt√≥re miary wymagajƒÖ prawid≈Çowego podzia≈Çu (ground truth). Te miary mierzƒÖ, jak dobrze przewidywane klastry pokrywajƒÖ siƒô z rzeczywistymi etykietami klas. Poni≈ºej znajdujƒÖ siƒô najwa≈ºniejsze miary wraz z przyk≈Çadami kodu.

Skorygowany Wska≈∫nik Randa (ARI)#

Skorygowany Wska≈∫nik Randa mierzy podobie≈Ñstwo miƒôdzy dwiema klasteryzacjami danych, analizujƒÖc wszystkie pary punkt√≥w i liczƒÖc te, kt√≥re sƒÖ poprawnie sklasyfikowane. Wska≈∫nik jest skorygowany o przypadkowe dopasowanie i przyjmuje warto≈õci od -1 (brak zgodno≈õci) do 1 (idealna zgodno≈õƒá).

Przyk≈Çad kodu:

# Wzorcowe etykiety

# Przewidywane etykiety klastr√≥w

# Obliczanie ARI
ari = adjusted_rand_score(y_true, y_pred)
print(f"Skorygowany Wska≈∫nik Randa (ARI): {ari:.2f}")

Znormalizowana Wzajemna Informacja (NMI)#

Znormalizowana Wzajemna Informacja ocenia ilo≈õƒá informacji wsp√≥lnej miƒôdzy rzeczywistymi etykietami a przewidywanymi klastrami. Wynik jest znormalizowany i mie≈õci siƒô w zakresie od 0 (brak wsp√≥lnej informacji) do 1 (idealna korelacja).

Przyk≈Çad kodu:

# Wzorcowe etykiety

# Obliczanie NMI

print(f"Znormalizowana Wzajemna Informacja (NMI): {nmi:.2f}")

Jednorodno≈õƒá, Kompletnosƒá i Miara V#

Jednorodno≈õƒá: Wszystkie klastry zawierajƒÖ wy≈ÇƒÖcznie punkty nale≈ºƒÖce do jednej klasy. Kompletno≈õƒá: Wszystkie punkty z danej klasy sƒÖ w tym samym klastrze. Miara V: ≈örednia harmoniczna jednorodno≈õci i kompletno≈õci. Przyk≈Çad kodu:

Podsumowanie#

Powy≈ºsze metryki dostarczajƒÖ cennych informacji na temat jako≈õci klasteryzacji, je≈õli dostƒôpne sƒÖ wzorcowe etykiety. Skorygowany Wska≈∫nik Randa (ARI) i Znormalizowana Wzajemna Informacja (NMI) sƒÖ szeroko stosowane do og√≥lnej oceny, podczas gdy Jednorodno≈õƒá, Kompletno≈õƒá i Miara V pozwalajƒÖ na bardziej szczeg√≥≈ÇowƒÖ analizƒô jako≈õci klastr√≥w. Wybieraj metrykƒô, kt√≥ra najlepiej pasuje do cel√≥w klasteryzacji.

import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_blobs
from sklearn.cluster import KMeans
from sklearn.metrics import adjusted_rand_score, normalized_mutual_info_score, homogeneity_score, completeness_score, v_measure_score

# Generowanie zbioru danych
X, y_true = make_blobs(n_samples=300, centers=4, cluster_std=1.0, random_state=42)

# Funkcja do wizualizacji wynik√≥w dla r√≥≈ºnych liczby klastr√≥w
def visualize_clustering_for_different_k():
    cluster_counts = [2, 4, 6, 8]  # R√≥≈ºne liczby klastr√≥w
    fig, axes = plt.subplots(2, 2, figsize=(15, 12))
    axes = axes.ravel()

    for i, k in enumerate(cluster_counts):
        # Klasteryzacja KMeans
        kmeans = KMeans(n_clusters=k, random_state=42)
        y_pred = kmeans.fit_predict(X)

        # Obliczanie miar jako≈õci klasteryzacji
        ari = adjusted_rand_score(y_true, y_pred)
        nmi = normalized_mutual_info_score(y_true, y_pred)
        homogeneity = homogeneity_score(y_true, y_pred)
        completeness = completeness_score(y_true, y_pred)
        v_measure = v_measure_score(y_true, y_pred)

        # Wizualizacja
        axes[i].scatter(X[:, 0], X[:, 1], c=y_pred, cmap='viridis', s=30)
        axes[i].scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], c='red', marker='x', s=100, label='Centroidy')
        axes[i].set_title(f"KMeans: {k} klastry\nARI: {ari:.2f}, NMI: {nmi:.2f}\nHomogeneity: {homogeneity:.2f}, Completeness: {completeness:.2f}, V-Measure: {v_measure:.2f}")
        axes[i].legend()

    plt.tight_layout()
    plt.show()

# Wywo≈Çanie funkcji
visualize_clustering_for_different_k()

import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score, davies_bouldin_score, calinski_harabasz_score
from sklearn.datasets import make_blobs
from ipywidgets import interact

# Define a function for clustering and visualization
def clustering_example(std_dev=0.6, n_clusters=4):
    # Generate synthetic data
    X, _ = make_blobs(n_samples=300, centers=4, cluster_std=std_dev, random_state=42)

    # Perform clustering
    kmeans = KMeans(n_clusters=n_clusters, random_state=0).fit(X)
    labels = kmeans.labels_

    # Compute clustering metrics
    sil_score = silhouette_score(X, labels)
    db_score = davies_bouldin_score(X, labels)
    ch_score = calinski_harabasz_score(X, labels)

    # Display metrics
    print("Clustering Metrics:")
    print(f"  Silhouette Score: {sil_score:.2f}")
    print(f"  Davies-Bouldin Index: {db_score:.2f}")
    print(f"  Calinski-Harabasz Index: {ch_score:.2f}")

    # Visualize clustering
    plt.figure(figsize=(8, 6))
    plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', s=50)
    plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], c='red', s=200, marker='X', label='Centroids')
    plt.title("K-Means Clustering")
    plt.xlabel("Feature 1")
    plt.ylabel("Feature 2")
    plt.legend()
    plt.show()

# Create interactive widget
interact(clustering_example, std_dev=(0.1, 4.0, 0.4), n_clusters=(2, 10, 1))

Redukcja wymiarowo≈õci#

Redukcja wymiarowo≈õci to inny typ uczenie nienadzorowanego, kt√≥rego zadaniem jest ekstrakcja najwa≈ºniejszych cech z danych przy redukcji (kompresji) tych danych. Jej celem jest:

Zmniejszenie wymiarowo≈õci danych
Wizualizacja danych (redukcja do 2 lub 3 wymiar√≥w)

PCA - Analiza G≈Ç√≥wnych Sk≈Çadowych#

PCA (Principal Component Analysis) to technika redukcji wymiarowo≈õci, kt√≥ra przekszta≈Çca dane do nowej przestrzeni z mniejszƒÖ liczbƒÖ wymiar√≥w, zachowujƒÖc jak najwiƒôcej informacji (wariancji danych).

Jak dzia≈Ça PCA:

1. Standaryzacja danych: Przekszta≈Çcenie danych do tej samej skali.

2. Obliczenie macierzy kowariancji: Wykrywanie relacji miƒôdzy zmiennymi.

3. Wyznaczenie warto≈õci w≈Çasnych i wektor√≥w w≈Çasnych: Identyfikacja kierunk√≥w najwiƒôkszej wariancji.

4. Projekcja na nowe osie: Przekszta≈Çcenie danych do przestrzeni g≈Ç√≥wnych sk≈Çadowych.

import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.datasets import make_blobs

# 1. Generowanie danych syntetycznych
X, _ = make_blobs(n_samples=300, centers=3, cluster_std=1.0, random_state=42)

# 5. Odzyskanie oryginalnych wsp√≥≈Çrzƒôdnych (rekonstrukcja)

# 6. Wizualizacja rekonstrukcji
